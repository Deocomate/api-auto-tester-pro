API Configuration Guide & Examples

To use the API Auto Tester, provide a JSON file containing an array of API endpoint configurations. Each object in the array represents a single API request.

JSON File Structure:

  • endpoint (string, required): The full URL of the API endpoint.
  • method (string, required): The HTTP method (e.g., "GET", "POST", "PUT", "DELETE", "PATCH").
  • data (object, optional): The request payload for methods like POST, PUT, PATCH.
  • headers (object, optional): Custom headers for the request (e.g., {"X-Custom-Header": "value"}). The Bearer token is added automatically if provided.
  • params (object, optional): URL query parameters (e.g., {"id": 1, "sort": "asc"}).
